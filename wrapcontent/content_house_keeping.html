Giao diện màn hình chính ➔ House Keeping ➔ Chọn các chức năng muốn dùng
Các chức năng của buồng phòng (House Keeping)

5.1 Báo phòng dơ (Set & Dirty)

Chọn loại (vacant/ occupied dirty) ➔ Start room ➔ End room ➔ Run

Chú thích
Vacant: Phòng trống
Occupied Dirty:Phòng đã dùng chưa dọn

5.2 Báo đang dọn (Set & cleaning)

Chọn phòng ➔ Run


5.3 Đánh dấu phòng chưa dọn (Set & Un-cleaned)

Chọn phòng muốn thao tác ➔ Run



5.4 Báo đã dọn (Set & Cleaned)

Chọn phòng muốn thao tác ➔ Run


Lưu ý: Có thể kéo để chọn nhiều phòng


5.5 Báo bảo trì (Set & Maintenance)

Chọn phòng ➔ Start Date ➔ End Date ➔ Reason ➔ Remarks ➔ Set / Unset ➔ Run



Chú thích
Start Room: Phòng bắt đầu
End Room:Phòng kết thúc
Start Date:Ngày bắt đầu
End Date:Ngày kết thúc
Reason:Lý do
Reamrks:Đánh giá nhận xét

Lưu ý: Khi bỏ chức năng bảo trì hoặc hết thời hạn thì phòng sẽ chuyển qua trạng thái phòng dơ
