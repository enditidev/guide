7.1 Đổi ngôn ngữ (Change Language)

  • Cách 1: Trên màn hình chính chọn System ➔ Change Language
  • Cách 2: Trên màn hình chính chọn nhấn F12 trên bàn phím

Lưu ý: Một số laptop sẽ nhấn Fn + F12 để đổi ngôn ngữ

7.2 Thay đổi mật khẩu (Change Password)

Giao diện màn hình chính chọn System ➔ Change Password ➔ Nhập mật khẩu cũ (old password) ➔ Nhập mật khẩu mới ➔ Xác nhận lại mật khẩu mới

7.3 Lịch sử chú thích (Log History)

Giao diện màn hình chính chọn System ➔ Log History

7.4 Phân quyền cho người sử dụng (Authority Group)

Giao diện màn hình chính chọn System ➔ Authority Users ➔ Chọn nhóm ➔ Chọn menu ➔ tích chọn các quyền

  • Phân quyền chi tiết

Chọn View Permissions ➔ Chọn User ➔ Chọn nhóm ➔ Tích vào các chức năng muốn chọn

Run: Xem
Add: Thêm
Delete: Xóa
Update: Sửa

Lưu ý: Muốn tích tất cả các quyền: Chọn Check All ➔ Apply

7.5 Tỷ giá ngoại tệ (Rate of Exchange)

Đặt tỉ giá: Giao diện màn hình chính ➔ System ➔ Rate of Exchange

Type Currency:Loại tiền tệ
Description:Mô tả
Rate Of Exchange:Tỷ giá
Currency exchange rates:Tỷ giá hoán đổi
Charge:Lệ phí
+Small Currency:Tiền tệ nhỏ

7.6 Đăng xuất (Log out)

  • Cách 1: Trên màn hình chính chọn System ➔ Log out
  • Cách 2: Trên màn hình chính chọn nhấn F11 trên bàn phím

Lưu ý: Một số laptop sẽ nhấn Fn + F11 để đăng xuất

7.7 Một số chức năng khác

  • Maximized Window (Phóng to màn hình): Trên màn hình chính chọn System ➔Maximized Window
  • Questions and Answers (Hỏi đáp)(: Trên màn hình chính chọn System ➔Questions and Answers
  • Message send to... (Gửi tin nhắn): Trên màn hình chính chọn System ➔Message send to...
  • Exit System (Thoát chương trình): Trên màn hình chính chọn System ➔Exit System

Lưu ý: Muốn thoát chương trình phải thoát bằng Exit System không thể nhấn dấu "x" trên màn hình để thoát
